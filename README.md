# Portuguese Vocabulary Site

A simple static site for studying Portuguese vocabulary.

## Structure

- `index.html` — main page
- `style.css` — styles
- `script.js` — quiz logic
- `data/vocab.json` — vocabulary data

## Study mode

- Show one Portuguese word each round
- Pick the correct Chinese meaning from 4 options
- Click **下一题** to continue

## Update workflow

1. Maintain the master vocabulary list in Excel.
2. Export or convert it to `data/vocab.json`.
3. Commit and push to GitHub.
4. GitHub Pages publishes the latest version.
